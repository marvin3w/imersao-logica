# [03-05] Relações, Oposição e Edução (2)

## Por que isso importa?

Alguém afirma: "Todo poeta é criativo." Você precisa reformular isso como "Algum criativo é poeta" para encaixar num silogismo. A operação é legítima? Depende da **educção**: a arte de derivar uma proposição de outra sem mudar a verdade. Sem educção, você reformula argumentos às cegas e invalida silogismos que estavam corretos, ou valida os que estavam errados.

A educção é inferência **imediata**: uma proposição entra, outra sai, sem termo médio. É o passo intermediário entre oposição e silogismo.

## Conceito Central

**Edução (ou equivalência imediata) é a operação pela qual se obtém, a partir de uma proposição, outra proposição equivalente em verdade, mediante conversão, obversão ou contraposição.**

## Conversão

Troca-se sujeito e predicado, respeitando regras de validade:

| Forma original | Conversão válida | Observação |
|---|---|---|
| **E** (Nenhum S é P) | Nenhum P é S | Conversão simples |
| **I** (Algum S é P) | Algum P é S | Conversão simples |
| **A** (Todo S é P) | Algum P é S | Conversão per accidens (particulariza) |
| **O** (Algum S não é P) | *Inválida* | Não se converte licitamente |

Por que **O** não converte? Porque "Algum S não é P" não garante "Algum P não é S" sem ambiguidade de distribuição.

Exemplo: "Todo professor é instrutor" converte per accidens em "Algum instrutor é professor" (não "Todo instrutor é professor").

## Obversão

Mantém o sujeito, nega o predicado e troca a qualidade (afirmativa ↔ negativa):

1. Negue o predicado (complemento lógico)
2. Troque a qualidade da proposição

| Original | Obversa |
|---|---|
| Todo S é P | Nenhum S é não-P |
| Nenhum S é P | Todo S é não-P |
| Algum S é P | Algum S não é não-P |
| Algum S não é P | Algum S é não-P |

"Toda rosa é vermelha" → "Nenhuma rosa é não-vermelha." Mesma informação, forma diferente.

## Contraposição

Combina obversão com conversão. Válida para **A** e **O**:

| Original | Contrapositiva |
|---|---|
| Todo S é P | Todo não-P é não-S |
| Algum S não é P | Algum não-P é S |

"Todo cidadão é contribuinte" → "Todo não-contribuinte é não-cidadão." Útil em debates jurídicos e contratuais: reformular a negação sem perder força lógica.

## Edução é / não é

| Edução é | Edução não é |
|---|---|
| Inferência imediata (1 → 1) | Silogismo (2 premissas → conclusão) |
| Preserva equivalência lógica (quando válida) | Garantia de verdade material |
| Ferramenta de reformulação | Retórica ou parafrase livre |

## Aplicação prática

Numa negociação: "Nenhum produto defeituoso será aceito." Por contraposição: "Todo produto aceito não é defeituoso." A segunda forma pode ser mais clara para redigir cláusulas. Sem educção, a reformulação pode alterar o sentido sem que ninguém perceba.
