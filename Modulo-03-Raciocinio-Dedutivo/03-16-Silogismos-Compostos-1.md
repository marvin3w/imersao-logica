## Por que isso importa?

Raramente paramos num silogismo único. Encadeamos: "Todo A é B. Todo B é C. Todo C é D. Logo…" Cada passo é válido isoladamente, mas o **encadeamento** exige regras próprias. **Polissilogismo** é a cadeia em que a conclusão de um silogismo vira premissa do próximo.

Sem polissilogismo, você valida cada degrau e erra na escada inteira.

## Conceito Central

**Polissilogismo é uma cadeia de silogismos onde a conclusão de um torna-se premissa do seguinte**, até chegar à conclusão final desejada.

## Estrutura

```
Silogismo 1: M₁-P / S-M₁ → conclusão₁ (S-P)
Silogismo 2: conclusão₁ como premissa + nova premissa → conclusão₂
…
Conclusão final
```

Exemplo encadeado:

1. Todo ser vivo é corpóreo. Toda planta é ser vivo. **Logo, toda planta é corpórea.**
2. Toda planta é corpórea. Toda roseira é planta. **Logo, toda roseira é corpórea.**

A conclusão do primeiro alimenta o segundo. M₁ = "ser vivo"; no segundo silogismo, "planta" funciona como termo médio.

## Requisitos do encadeamento

| Requisito | Por quê |
|---|---|
| Cada silogismo individual válido | Um elo inválido invalida a cadeia |
| Termos usados com mesma suposição | Homonímia entre elos quebra a cadeia |
| Conclusão parcial respeitada em cada passo | Não "pular" de particular para universal no meio |

Pense na cadeia como elos de corrente: cada elo aguenta tensão se for íntegro; a corrente quebra no elo mais fraco.

## Polissilogismo vs. sorites

| Polissilogismo | Sorites |
|---|---|
| Cadeia explícita de silogismos | Forma abreviada (uma sequência de premissas) |
| Todas premissas e conclusões intermediárias visíveis | Conclusões intermediárias suprimidas |
| Análise passo a passo | Análise por regras do sorites ([03-17](03-17-Silogismos-Compostos-2.md)) |

## Aplicação prática

Numa argumentação sobre saúde: "Todo exercício regular melhora resistência. Toda caminhada diária é exercício regular. Logo, caminhada melhora resistência. Toda caminhada melhora resistência. João caminha diariamente. Logo, João melhora resistência." Dois silogismos encadeados. Verifique cada um separadamente antes de aceitar a conclusão sobre João.

## Diagrama do polissilogismo

```
Premissa₁ + Premissa₂ → Conclusão₁ (= Premissa₃)
Premissa₃ + Premissa₄ → Conclusão₂
…
Conclusão final
```

Cada seta é um silogismo completo com suas regras. Pular a verificação de um elo intermediário é aceitar a conclusão final por **autoridade da cadeia**, não por validade demonstrada.

## Quando usar polissilogismo explícito

Em textos longos (pareceres, petições, demonstrações), expandir o polissilogismo torna o argumento **auditável**. Em conversa, o sorites ([03-17](03-17-Silogismos-Compostos-2.md)) abrevia. Escolha explícita entre forma expandida e abreviada depende do público: especialistas exigem elos visíveis; leigos toleram abreviação se a conclusão for prudente.
