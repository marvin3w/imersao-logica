## O que é este material?

Um guia progressivo de Lógica Clássica, organizado em torno das **Três Operações do Intelecto**: Concepção, Juízo e Raciocínio. Cada operação corresponde a uma capacidade fundamental do pensamento humano, e dominá-las na ordem certa é o que permite pensar com clareza e argumentar com rigor.

## Mapa de progressão

```
M00: O que é Lógica? Para que serve?
 │
 ▼
M01: CONCEPÇÃO ─── Como formamos conceitos?
 │   Conceito, Definição, Divisão, Predicamentos, Termos
 │
 ▼
M02: JUÍZO ─── Como afirmamos ou negamos algo?
 │   Proposição, Tipos de proposições, Verdade, Formas AEIO
 │
 ▼
M03: RACIOCÍNIO DEDUTIVO ─── Como tiramos conclusões válidas?
 │   Silogismo, Figuras, Modos, Princípios, Falácias
 │
 ▼
M04: INDUÇÃO E CRÍTICA ─── Como generalizamos a partir de casos?
     Argumento indutivo, Princípios da indução
```

## Por que esta ordem?

A Lógica clássica segue uma progressão natural: primeiro você precisa entender os **blocos** (conceitos), depois como eles se **combinam** (proposições), e por fim como se **encadeiam** para produzir conhecimento novo (raciocínio). Pular etapas gera confusão: não dá para entender silogismos sem dominar proposições, nem proposições sem dominar conceitos.
