# Imersão em Lógica Clássica

Corpo de conteúdo autossuficiente sobre **Lógica Clássica** (tradição aristotélica-tomista), reestruturado para assimilação progressiva com didática clara.

## Entrada

| | |
|---|---|
| **Mapa do curso** | [Panorama Geral](00-Panorama-Geral.md) |
| **Primeira sessão** | [M00 — Fundamentos](Modulo-00-Fundamentos/00-00-Visao-Geral.md) |

A progressão entre sessões é controlada **dentro de cada documento** (barra no topo + seção Conexões). Não é necessário memorizar a ordem dos módulos aqui.

## Origem

Material produzido a partir de um curso de Lógica em vídeo (53 aulas), com reescrita completa usando padrões didáticos que priorizam:
- Motivação antes de teoria ("por que isso importa?")
- Progressão do simples ao complexo
- Exemplos concretos antes de definições formais
- Contraste explícito ("o que é vs. o que não é")

Regras editoriais: [Regras-Editoriais-Logica.md](Regras-Editoriais-Logica.md)
