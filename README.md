# Imersão em Lógica Clássica

Corpo de conteúdo autossuficiente sobre **Lógica Clássica** (tradição aristotélica-tomista), reestruturado para assimilação progressiva com didática clara.

## Estrutura

O material segue a divisão clássica das **Três Operações do Intelecto**:

| Módulo | Tema | Aulas-fonte |
|--------|------|-------------|
| [M00](Modulo-00-Fundamentos/) | Fundamentos e Natureza da Lógica | 03 |
| [M01](Modulo-01-Concepcao/) | Primeira Operação: Concepção (Conceitos e Definições) | 04-13 |
| [M02](Modulo-02-Juizo/) | Segunda Operação: Juízo (Proposições) | 14-36 |
| [M03](Modulo-03-Raciocinio-Dedutivo/) | Terceira Operação: Raciocínio Dedutivo (Silogismo) | 37-53 |
| [M04](Modulo-04-Inducao-Critica/) | Raciocínio Indutivo e Crítica Argumentativa | 54-55 |

## Como usar

1. Comece pelo **Módulo 00** para entender o que é Lógica e por que ela importa
2. Siga a sequência M01 → M02 → M03 → M04 (cada módulo pressupõe o anterior)
3. Dentro de cada módulo, os documentos estão ordenados por progressão conceitual
4. Use as seções "Auto-reflexão" para verificar sua compreensão antes de avançar
5. Analogias com situações do cotidiano e referências a clássicos (Schopenhauer, Aristóteles) aparecem ao longo do texto como apoio, sem pressupor que você os conheça

## Origem

Material produzido a partir de um curso de Lógica em vídeo (53 aulas), com reescrita completa usando padrões didáticos que priorizam:
- Motivação antes de teoria ("por que isso importa?")
- Progressão do simples ao complexo
- Exemplos concretos antes de definições formais
- Contraste explícito ("o que é vs. o que não é")
